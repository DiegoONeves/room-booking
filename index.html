Como rodar a aplicação

Para rodar a aplicação você precisa do Visual Studio 2013 ou 2015 (Qualquer versão, incluindo a Community) e uma instância do SQL Express rodando na sua máquina, ou em algum outro local com permissão para criar um novo banco de dados.

Clone o repositório:

$ git clone https://github.com/andrebaltieri/room-booking.git

Altere a Connection String RoomBookingConnectionString na pasta "1 - API/RoomBooking.Api/Web.Config", apontando para o seu servidor SQL Server.

Clique com o botão direito sobre o projeto "1 - Presentation/RoomBooking.Api" e selecione a opção "Set as Startup Project". Em seguida, execute a aplicação utilizando o comando "CTRL+F5".

Repita o procedimento acima com a aplicação "1 - Presentation/RoomBooking.Web".
